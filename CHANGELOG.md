## [1.0.0] - 2026-07-16

### Added
- High-performance Go backend API utilizing Gin, pgx, and gocron.
- Multiphase ingestion pipeline for parsing, filtering, and scoring RSS feeds.
- React 19 SPA dashboard featuring Shadcn components, Radix UI primitives, and TanStack query hooks.
- Python-based embedding sidecar for free, local semantic duplicate detection, ranking, and search.
- Multi-container local orchestration configuration using Docker Compose.
- Complete pnpm workspace integration.
